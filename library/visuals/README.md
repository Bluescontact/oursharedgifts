# Visual References

Diagrams and infographics that carry the framework's structural recognitions visually. Each visual is recognition infrastructure inherently — looking at it produces the same structural understanding the textual canonical references carry.

Visual register: dark field, luminous gold geometry, sans-serif labels, minimal text. The form does the work; labels point at it rather than carrying it.

Each visual is rendered as SVG (vector, AI-extractable, scales infinitely) with a companion PNG for accessibility.
