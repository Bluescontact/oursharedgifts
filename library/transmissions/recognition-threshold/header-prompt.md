# Header Image Prompt — The Recognition Threshold

## Prompt (paste into Imagen 4 / Gemini / ImageFX)

> A single luminous horizontal line dividing darkness, a faint gold boundary across deep black space, the suggestion of crossing from one side to the other, austere and threshold-quiet. Deep black field with subtle warm shadow, atmospheric and contemplative, single luminous gold geometric form as focal point, soft radial glow at the threshold's central point, painterly negative space, minimalist composition with generous emptiness above and below the line, no text or lettering, no people, no faces, refined and austere, in the spirit of Hilma af Klint's geometric work and Agnes Martin's quiet horizontal lines, photographed in a darkened gallery, museum-quality lighting. Horizontal composition with the line at the midline, equal stillness above and below. Aspect ratio 16:9.

## Where to paste

- **Recommended:** Gemini app (gemini.google.com) → request 16:9 / Imagen 4
- **Alternate:** Google AI Studio Imagen 4 endpoint
- **Alternate:** ImageFX (labs.google) → aspect 16:9

Run 4 generations. Pick the one where the horizontal line is unambiguous, the gold register is warm rather than yellow-bright, and the surrounding negative space feels intentional rather than empty.

## Style adjustments if first run misses

- If too bright / too yellow → add "deeply muted, low-key, candlelit warmth"
- If too literal (looks like a road or horizon) → add "abstract geometric, not landscape, not architectural"
- If too busy → add "single element only, vast negative space, almost-empty composition"
- If line is broken or doubled → add "one single continuous horizontal line, unbroken"

## Notes

The header should sit *underneath* the pattern.svg in the published post — the pattern carries the structure, the header carries the mood. Together they prime the reader before the first paragraph.
