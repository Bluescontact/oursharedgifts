# Pull-Quotes — Twelve Failure-Mode Patterns

Standalone share-cards. Dark field, warm bone text, no padding, no emoji.

---

## 1

A failure pattern names which vertex is missing. The corrective is to restore the vertex, not to fight the pattern.

---

## 2

The breakage is not the problem. It is the symptom of a missing vertex.

---

## 3

A Transaction Trap addressed with more documentation deepens the trap. A Burnout Cycle addressed by working harder accelerates the breakage. The surface fix is never the corrective.

---

## 4

One person is the runtime of the system. The system is not fragile because of them — it is fragile *as* them.

---

## 5

A system that has failed the same way three times has already named which vertex is structurally weak. The third recurrence is the diagnostic.
