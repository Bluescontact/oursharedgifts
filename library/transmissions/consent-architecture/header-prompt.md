## Notes
- Geometry: vertex-mapped / crossing axes (the Wheel of Consent as two principles producing four positions).
- Mood: austere, structural, threshold-quiet, clarifying, load-bearing.
- The crossing point should be the brightest element — it reads as the moment consent holds. Keep the four quadrant nodes dim and balanced; none should dominate (no position is morally higher than another).
- Run 4 generations. Pick the one where the cross stays quiet and the negative space holds. If a generation introduces any color beyond the gold spectrum, discard it — do not weaken the locked aesthetic block, adjust only the subject-and-mood slot.
