Aquaponic Greenhouse

STATUS: Active Development
View on GitHub → Support This Project

1-acre demonstration site utilizing holistic and systems-based design for rapid biodiversity generation, permanent pond infrastructure, and integrated food forest. Combining aquaponics production with regenerative land development in New Mexico high desert conditions.

Scale
1 acre demonstration site
Location
New Mexico
Design Approach
Holistic, systems-based
Primary Systems
Aquaponics + Food Forest

Design Principles

Rapid Biodiversity Generation

  • Stacking functions within limited space
  • Permanent water features as ecosystem anchors
  • Integration of aquatic and terrestrial food production
  • Native plant integration with productive species

Systems Integration

  • Closed-loop nutrient cycling between fish and plants
  • Greywater treatment through constructed wetlands
  • Thermal mass from pond moderating greenhouse temperatures
  • Excess production feeding food forest establishment

Current Phase

Infrastructure buildout in progress. Establishing pond system, greenhouse foundation, and perimeter food forest plantings. Documentation includes site assessment, water system design, greenhouse construction details, and aquaponics plumbing specifications.

Why this matters: Demonstrates integration of intensive food production with regenerative land practices. Proof of concept for rapid establishment of productive ecosystems in challenging climates. Documentation enables replication by others working in similar conditions.

Mobile Hexayurt Platform

STATUS: Design Refinement
View on GitHub → Support This Project

Deployable shelter system based on hexayurt geometry with mobile platform integration. Designed for rapid deployment, thermal efficiency in extreme conditions, and minimal-tool assembly. Combines proven hexayurt design with custom mounting system for vehicle-based mobility.

Base Design
Hexayurt geometry
Mobility
Vehicle-mounted platform
Assembly Time
~2 hours (2 people)
Tools Required
Minimal (tape, knife)

Design Principles

Rapid Deployment

  • Pre-cut panels stored flat for transport
  • Tape-based assembly (no fasteners required)
  • Single vehicle carries complete system
  • Two-person setup in under two hours

Thermal Performance

  • Reflective exterior for solar gain management
  • Insulated panel construction
  • Hexagonal geometry optimizes volume-to-surface ratio
  • Natural ventilation through roof geometry

Current Phase

Refining mobile platform design and panel mounting system. Testing material specifications for durability under transport conditions. Developing documentation for replication including cut patterns, assembly sequence, and platform fabrication details.

Why this matters: Provides rapidly deployable shelter solution using accessible materials and minimal tools. Addresses nomadic living, emergency response, and temporary work camp applications. Open documentation enables local adaptation to regional materials and conditions.

Bus Mobile Crane Platform

STATUS: Infrastructure Buildout
View on GitHub → Support This Project

Conversion of 1995 school bus into mobile crane platform with integrated workshop, power systems, and materials transport capability. Designed to support remote construction projects while providing mobile living quarters. Eight years of elevator construction experience applied to mobile infrastructure.

Base Vehicle
1995 School Bus
Primary Function
Mobile crane + workshop
Power System
30kWh battery + solar
Lifting Capacity
Design target: 2 tons

Design Principles

Integrated Capability

  • Crane platform for heavy lifting at remote sites
  • Workshop space with tool storage and work surfaces
  • Materials transport (lumber, panels, supplies)
  • Living quarters for extended project work

Power Independence

  • 30kWh lithium battery bank
  • Solar charging system
  • Shore power integration when available
  • Powers tools, living systems, and auxiliary equipment

Current Phase

Completing electrical system integration and structural reinforcement for crane mounting. Developing documentation for power system design, structural modifications, and crane integration. Focus on safety systems and load distribution analysis.

Why this matters: Demonstrates integration of heavy construction capability with mobile living platform. Addresses challenges of remote building projects where crane rental isn't viable. Documentation enables others to develop mobile construction platforms suited to their needs.

Solar Biochar Toilet

STATUS: Detached Processing Container + Heliostat
View on GitHub → Support This Project

Solar-powered pyrolysis system converting human waste into sterile biochar through anaerobic thermal decomposition. Heliostat concentrates solar energy onto processing container, achieving 400-700°C temperatures for complete pathogen destruction and carbon sequestration. Closes nutrient loop while eliminating waste stream.

Processing Method
Solar pyrolysis (anaerobic)
Container Size
5-gallon cast iron
Processing Temperature
400-700°C
Batch Frequency
Weekly/bi-weekly

Design Principles

Complete Resource Recovery

  • Biochar output: sterile carbon soil amendment
  • Syngas capture: combustible gas (CO, H₂, CH₄) for cooking
  • Urine separation: nitrogen-rich fertilizer (separate system)
  • Zero waste stream—complete nutrient loop closure

Solar Concentration System

  • Heliostat tracks sun position throughout processing
  • Concentrates solar energy onto stationary processing container
  • Cast iron vessel provides thermal mass and even heating
  • Detached processing—staging area separate from collection point

Current Phase

Developing heliostat tracking system and processing container design. Testing thermal requirements for 5-gallon batch pyrolysis using cast iron vessel. Refining syngas capture and storage approach. Documentation focuses on solar concentration geometry, temperature monitoring, and safety protocols for anaerobic pyrolysis.

Why this matters: Addresses sanitation in off-grid contexts without creating waste streams or requiring infrastructure. Converts liability (human waste) into assets (biochar, syngas, fertilizer). Solar-powered processing eliminates fuel requirements. Documentation enables replication for remote homesteads, disaster response, and resource-constrained communities.

  • Ask questions - Stuck on implementation? Open a discussion.
